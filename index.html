Novos Negócios - Gerencia a visita de vendedores aos clientes que visita

Desenvolvido para uma empresa que entre as suas atribuições visitava clientes em potencial em busca de novos negócios. A idéia era que o aplicativo facilitasse a vida do vendedor na hora de anotar informações sobre o local que visitou assim como facilitar a análise e controle dos dados mais tarde no Excel. A partir de uma tela no mapa onde com um clique é possível marcar a posição atual com dados do GPS ou buscar um endereço assim como facilmente adicionar detalhes sobre o cliente. Após coletar os dados era possível exibí-los na tela do celular em pontos no mapa ou em tabela e exportá-los para o Google Drive em quatro formatos diferentes. Era possível também no aplicativo criar um novo banco de dados com informações que o vendedor achasse mais pertinente inclusive com campos personalizados.

Biblioteca - Gerencia e pesquisa livros de uma biblioteca universitária

Desenvolvido para que os alunos de uma universidade pudessem renovar o impréstimo de livros através de seus smartphones para que não ocorresse atraso na devolução e pudessem pesquisar o acervo de livros da biblioteca conforme banco de dados online, o aplicativo dava opção de login que por um parser de html se conectava ao servidor da biblioteca e dava as opções de renovação ou pesquisa de livros. O usuário também podia criar um alarme que gerava notificação no smartphone no dia em que o livro precisava ser devolvido.

Goal Timer - Um temporizador de atividade para alcançar metas

Um temporizador que te ajuda a alcançar uma meta, caso sua meta seja correr 30 minutos mas não possui o condicionamento físico para isso, você define no aplicativo tempo total (sua meta), tempo de repetição (quanto consegue correr sem parar) e quantidade de repetições. E conforme for se acostumando com os exercícios você pode aumentar o tempo de repetição o que consequentemente diminui o tempo de pausa (descanso) e alcançar seu objetivo. Serve para qualquer tipo de atividade como estudar, focar no trabalho, fazer exercícios, etc. O aplicativo foi escrito em Kotlin e o temporizador roda em segundo plano através da biblioteca RxJava, também possui botões de Pausa e Parada para melhor controle de suas atividades. Link para o repositório GitHub com o código fonte

Método de McCabe Thiele - Solução gráfica do método

O método de McCabe Thiele é bastante utilizado no aprendizado de operações unitárias na Engenharia Química para encontrar o número de pratos teóricos em uma destilação binária para alcançar um nível desejado de pureza. E sua análise, apesar de simplificada por fazer diversas suposições, auxilia a compreensão de como as diferentes variáveis do processo e o diagrama de equilíbrio influenciam o processo de destilações binárias. No passado esse algorítmo já foi utilizado por mim em um projeto comercial mas como ele não existe em código aberto para Android, foi reescrito por mim em Kotlin utilizando uma biblioteca de gráficos e o algorítmo foi disponibilizado gratuitamente. Link para o repositório GitHub com o código fonte

Reddit Just Show Me Images - Um aplicativo para navegar no site de mídia social Reddit

Existem diversos aplicativos para Android que navegam no Reddit, um dos maiores sites de entretenimento e notícias sociais, porém todos seguem a mesma linha de lista com links de postagens, daí surgiu a idéia de um aplicativo que navegue no Reddit através de imagens em uma tela de grades, basta o usuário entrar em qual página (subreddit) ele quer entrar e é mostrado uma grande quantidade de imagens daquele assunto sem enrolações. O aplicativo recolhe os dados do API do Reddit através das biblioteca Retrofit e RxJava e ligam esses dados a um Recycler View. Link para o repositório GitHub com o código fonte
